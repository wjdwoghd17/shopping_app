# 🛒 쇼핑 리스트 앱

아이템 추가·삭제·체크 기능을 갖춘 심플한 웹 쇼핑 리스트입니다.

## 실행 방법

`index.html` 파일을 브라우저에서 바로 열면 됩니다. 별도의 서버나 빌드 과정이 필요 없습니다.

## 기능

- **추가** — 텍스트 입력 후 `Enter` 또는 `+` 버튼
- **체크** — 체크박스 클릭 시 취소선 표시
- **삭제** — 각 항목 우측 `✕` 버튼으로 개별 삭제
- **일괄 삭제** — 완료 항목만 한 번에 삭제
- **데이터 유지** — `localStorage`에 저장되어 새로고침 후에도 유지

## 테스트

Playwright를 이용한 E2E 테스트 포함:

```bash
npm install
node test.js
```
